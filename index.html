about hub

this is the main hub project. i need to get enough of it working that i'd be happy living with it as the primary hub. until then this is more of a backup or dev hub project.

ideally multiple hubs should be able to exist on the same network. they should identify each other and have some logic for deciding which one is the primary hub.

this could also end up being the universal base for all the Null devices.

automation

with the Null Lights plugin Null can control the lights and fans that are plugged into them. with the Null Weather plugin local sunrise and sunset times can be used to determine when lights should be one.

lights can be snoozed after set amounts of time. they can also be turned on when lights in a neighboring room are on; and snooze after those lights have been off for a short while.
